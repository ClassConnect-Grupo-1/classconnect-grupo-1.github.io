Backoffice User Guide

An overview of how our internal admin panel operates

Overview

The backoffice system is designed to help administrators and internal staff manage platform operations efficiently.

Core Features

  • User Management: View and update users. An admin can change any user's role (teacher or student) and can block or unblock a user.
  • Metrics: Watch metrics diagrams about logins, notifications or course approvals per user and day. To do so, you have to select a range of dates.
