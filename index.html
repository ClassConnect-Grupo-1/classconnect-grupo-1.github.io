ClassConnect User Guide

An overview of how our backoffice and mobile app operates

Overview

ClassConnect is a mobile app where teachers and students can join/create courses to improve their learning experience.

Backoffice

The backoffice system is designed to help administrators and internal staff manage platform operations easily.

  • User Management: View and update users. An admin can change any user's role (teacher or student) and can block or unblock a user.
  • Metrics: View metric diagrams about logins, notifications, or course approvals per user and per day. To do this, select a range of dates from the filter panel.

Mobile App Guide

The mobile app allows users to interact with the platform on the go. It provides access to core features with a simplified and responsive interface.

  • Login: Users log in using the same credentials as the desktop platform. Biometric login is also available on supported devices.
  • Notifications: Push notifications alert users to course updates, approvals, and important messages.
  • Course Access: Teachers and students can view and manage course content, check schedules, and track progress directly from the app.
  • Offline Mode: Users can download materials and work offline. Sync occurs automatically once the connection is restored.
For the best experience, ensure the app is updated to the latest version via the App Store or Google Play.
